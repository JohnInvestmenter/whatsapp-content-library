## Method 2: Canva Docs to Deck (Convert Document to Slides)

### Step 1: Create a Canva Doc
1. Go to Canva
2. Click **"Create a design"**
3. Select **"Doc"**

### Step 2: Use AI to Generate Content
1. In the Doc, click the **"+"** button
2. Select **"Magic Write"**
3. Paste your prompt from `CANVA_PROMPT.txt`
4. Let AI generate the document

### Step 3: Convert to Presentation
1. Click **"File"** menu
2. Select **"Create presentation from Doc"**
3. Canva automatically creates slides from headings
4. Review and customize

## Method 3: Manual Method (If AI Tools Not Available)

### Step 1: Start with Template
1. Go to Canva
2. Search for **"Business Presentation"** templates
3. Choose a modern, professional template
4. Select one with 30+ slides

### Step 2: Customize Slide by Slide
1. Use the slide breakdown in `CANVA_PROMPT.txt`
2. Replace template text with content from each slide
3. Adjust layouts as described
4. Add icons and images

### Step 3: Maintain Consistency
1. Use the color palette mentioned:
   - Primary: #8b5cf6 (Purple)
   - Secondary: #3b82f6 (Blue)
   - Accent: #10b981 (Green)
2. Use consistent fonts (Montserrat or Inter)
3. Keep spacing and alignment uniform

---

## Method 4: Bulk Create (For Advanced Users)

### Step 1: Prepare Data
1. Create a CSV file with slide data
2. Format: Title, Content, Visual, Layout

### Step 2: Use Bulk Create
1. In Canva, go to **"Apps"**
2. Search for **"Bulk Create"**
3. Upload your CSV
4. Map fields to slide elements
5. Generate all slides at once

## Pro Tips for Best Results

### Before Starting:
- [ ] Have your logo ready (PNG with transparent background)
- [ ] Prepare 5-10 screenshots from your actual Content Library
- [ ] Get team photos or stock photos for testimonials
- [ ] Have contact information ready

### Customization Checklist:
- [ ] Replace "[your-email@company.com]" with real email
- [ ] Add your logo to master slide
- [ ] Update color scheme to match your brand (optional)
- [ ] Add real screenshots instead of mockups
- [ ] Include actual usage statistics if available
- [ ] Add QR code for easy scheduling link
- [ ] Include company name in footer

### Design Best Practices:
- Keep text minimal (6-7 lines max per slide)
- Use large, readable fonts (min 24pt)
- High contrast text on backgrounds
- One main idea per slide
- Use visuals to support text, not replace it
- Consistent icon style throughout

## Canva Features to Use

### Essential Tools:
1. **Elements** (Left sidebar)
   - Search for icons: "whatsapp", "chat", "AI", "stats"
   - Add shapes for callouts
   - Use lines and arrows for flow

2. **Text**
   - Heading: 44-54pt
   - Subheading: 28-36pt
   - Body: 18-24pt

3. **Photos**

## Troubleshooting

### If AI Generation Fails:
1. Break the prompt into smaller sections
2. Generate 5-10 slides at a time
3. Use the manual method as backup

### If Design Looks Off:
1. Start with a professional template
2. Apply your content to existing layouts
3. Keep original design structure

### If Text Doesn't Fit:
1. Reduce font size (but not below 18pt)
2. Use bullet points instead of paragraphs
3. Split into multiple slides
